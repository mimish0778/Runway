# RUNWAY

個人利用を目的として作成した、シンプルなフォーカス管理 Web アプリです。

Play : https://mimish0778.github.io/Runway/

---

## 使い方

### 1. 起動
ブラウザでページを開くと、  
`STANDING BY` 状態で待機しています。

画面中央の `START` ボタンを押して開始します。

---

### 2. Idle（準備フェーズ）
- 1分間のカウントダウンが始まります  
- これから作業に入るための「準備時間」です  
- 何もしなくても問題ありません

カウントが終了すると、次の行動を選択できます。

---

### 3. 行動の選択
Idle 終了後、以下のどちらかを選びます。

- **KEEP GOING**  
  → Work（作業フェーズ）へ進みます

- **TAKE A BREAK**  
  → Break（休憩フェーズ）へ進みます

---

### 4. Work（作業フェーズ）
- 作業時間の計測が自動で始まります  
- 時間は継続的にカウントされ、進捗が表示されます  
- 区切りたいタイミングで `TAKE A BREAK` を押します

Work を終了すると、作業時間が記録されます。

---

### 5. Break（休憩フェーズ）
- 1分間の休憩カウントダウンが始まります  
- カウント終了、または `SKIP` を押すと終了します

終了後は初期状態に戻り、再度 `START` できます。

---

### 6. 表示される情報について
- **SESSIONS**：完了した Work の回数  
- **TOTAL TIME**：これまでの作業累計時間  
- **LOG**：直近の Idle / Work / Break の履歴  

すべてブラウザ上で完結し、外部へ送信されることはありません。
